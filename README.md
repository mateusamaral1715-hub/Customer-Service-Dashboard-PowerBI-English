# Customer-Service-Dashboard-PowerBI-English

📊 Customer Service (SAC) Performance Dashboard
This repository contains a professional Power BI Dashboard designed to optimize Customer Service (SAC) operations. As a Bachelor of Business Administration transitioning into Data Analytics, I developed this project to demonstrate how data-driven insights can transform operational efficiency and customer satisfaction.

🎯 Project Overview
In modern business, managing support teams without clear metrics leads to bottlenecks and poor customer experience. This dashboard provides a 360-degree view of SAC performance, allowing managers to identify trends and improve service quality.

🛠️ Tech Stack & Skills
Power BI: Data visualization and dashboard architecture.

SQL: Used for data extraction, queries, and joins.

DAX (Data Analysis Expressions): Development of complex measures and calculated columns.

Power Query (M): Extensive ETL (Extract, Transform, Load) processes for data cleaning.

Business Intelligence: Strategic KPI tracking and reporting.

📈 Key Performance Indicators (KPIs)
The dashboard monitors critical metrics for any support operation:

Average Handle Time (AHT / TMA): Measuring team productivity and efficiency.

Customer Satisfaction Score (CSAT): Tracking the quality of the service provided.

Ticket Volume: Analyzing demand trends over time.

Resolution Rates: Evaluating the effectiveness of the support agents.

⚙️ Development Process
Data Extraction: Used SQL to pull relevant information from databases.

ETL: Performed data cleaning and transformation using Power Query to ensure data integrity.

Modeling: Built a robust star-schema data model.

Calculations: Authored advanced DAX measures to create dynamic time-intelligence and performance indicators.

Visualization: Designed an intuitive interface focusing on User Experience (UX) and actionable business insights.
